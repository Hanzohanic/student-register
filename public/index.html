Welcome to the Student portal

How it works

This is a student management system, its for managing different aspects of the administrative process at a educational institution. The features offered include the ability for the school administration to register students. It also includes an interface for students to be able to see the tuition amount they need to pay for that particular school term as well as be able to pay using celo.
